## 🔐 Auth System I Built — No Backend Needed

```
User registers → OTP sent to real email (EmailJS API)
              → OTP verified (stored in IndexedDB, expires 10 min)
              → Password created (SHA-256 hashed, Web Crypto API)
              → Session token saved (localStorage + IndexedDB)
              → Page reload = auto login (7-day token) ✅

Security highlights:
  ✅ Passwords never stored in plain text — SHA-256 only
  ✅ OTP expires in 10 minutes
  ✅ Max 5 wrong attempts before lockout
  ✅ Crypto-random tokens via window.crypto (not Math.random)
  ✅ Sessions auto-expire after 7 days
```
