## 🔐 Auth System I Built (No Backend!)
 
```
FLOW:  Register → Email OTP → Verify → Set Password → Dashboard
 
TECH USED:
┌─────────────────────────────────────────────────────────┐
│  IndexedDB       → Stores users, sessions, OTPs         │
│                    (browser database, persists forever)  │
│                                                         │
│  Web Crypto API  → SHA-256 password hashing             │
│                    (browser-native, zero npm deps)       │
│                                                         │
│  EmailJS         → Sends real OTP to user's Gmail       │
│                    (no Node.js server needed!)           │
│                                                         │
│  localStorage    → Stores session token                 │
│                    (auto-restores login on page reload)  │
└─────────────────────────────────────────────────────────┘
 
SECURITY:
✅ Passwords never stored in plain text
✅ OTP expires in 10 minutes
✅ Max 5 wrong OTP attempts
✅ Session expires after 7 days
✅ Crypto-random tokens (not Math.random)
```
